# Contributing to Equipmentals.pk

First off, thank you for considering contributing to Equipmentals.pk. It's people like you that make open source such a great community!

## 1. Where do I go from here?

If you've noticed a bug or have a request for a new feature, please open an issue in the issue tracker. If you'd like to work on a feature, feel free to fork this repository and submit a pull request!

## 2. Forking & Pull Requests

1. **Fork** the repository on GitHub.
2. **Clone** the project to your own machine.
3. **Commit** changes to your own branch.
4. **Push** your work back up to your fork.
5. Submit a **Pull Request** so that changes can be reviewed.

## 3. Getting Started
Make sure to follow the local environmental setups listed thoroughly in the `README.md`. Your PRs must ensure that no build or dependency breaks in either the Frontend or Backend environments. Please lint your code effectively and remove console logs before committing.

## 4. Code of Conduct
Please be respectful to other contributors. Harassment, derogatory comments, and unprofessional conduct will not be tolerated.

Thank you again for contributing!
